# 🖥️ Edge Model Inference (雲端運算與邊緣運算應用 - 模型推論合集)

## 📖 專案簡介
集合 LeNet、ResNet、EfficientDet 模型的完整實驗流程，涵蓋 **PC 訓練 → Edge 部署**。

## ⚙️ 功能特色
- LeNet：MNIST 手寫辨識
- ResNet：影像分類
- EfficientDet：物件偵測
- Edge 部署測試（DevBoard）

## 🛠️ 技術架構
- **框架**：TensorFlow / PyTorch
- **Edge 平台**：Coral Dev Board
- **模型格式**：TFLite

## 📊 成果展示
- LeNet 成功辨識 MNIST 圖片
- ResNet 部署於 Edge，完成即時分類
- EfficientDet 成功於 DevBoard 偵測物件
